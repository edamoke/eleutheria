# Eleutheria Operational Control Interface

This is the frontend dashboard for the **Eleutheria Self-Sufficient Server Network**. It provides real-time monitoring and governance capabilities for a decentralized network of autonomous edge computing units.

## Features
- **Sovereign Node Monitoring**: View real-time status of nodes, including NPU acceleration and local AI inference metrics.
- **Intent Governance**: Submit and monitor cryptographic intents for network-wide actions.
- **Edge Analytics**: Track real-time production and telemetry data from localized edge units.
- **Financial Transparency**: Monitor capital allocation and revenue flows anchored on-chain.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technical Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Connectivity**: Axios for MION service interaction
